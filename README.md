# Temperature Converter Website 🌡️

### 👨‍💻 Developer Details
* **Name:** Ishant Khare
* **Domain:** Web Development
* **Company:** Oasis Infobyte
### 🎯 Objective
To create an interactive web application that instantly converts temperatures between Celsius, Fahrenheit, and Kelvin.

### 🛠️ Tools Used
* HTML5
* CSS3
* JavaScript

### 📝 Steps Performed
1. Designed a clean input interface using HTML and styled it with CSS.
2. Implemented JavaScript event listeners to capture user input and button clicks in real-time.
3. Wrote mathematical functions to dynamically calculate the conversions and manipulate the DOM to display the results.

### 💡 Outcome
A fully functional, interactive web tool that provides highly accurate and instant temperature conversions directly in the browser.
